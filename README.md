# Fair Care Pharmacy Website

A responsive business website developed for a local medical store as a freelance project.

The website provides store information, services, contact details, and location access for customers. It is designed to help customers quickly find medicines, contact the store, and visit the shop.

---

## Live Website

Current Live Demo: https://fair-care-website.vercel.app

*A custom domain will be connected soon.*

---

## Features

* Responsive design (mobile, tablet, desktop)
* Clean modern UI
* Store services section
* Contact section with phone access
* Google Maps location
* WhatsApp quick contact button
* Fast loading and optimized images

---

## Tech Stack

**Frontend**

* Next.js (React Framework)
* TypeScript
* Tailwind CSS
* Shadcn UI Components

**Tools**

* Git & GitHub (version control)
* Vercel (deployment & hosting)

---

## Project Purpose

This project was developed as a freelance website for a local pharmacy to establish an online presence and help customers easily access store information and contact details.

---

## Installation (Run Locally)

1. Clone the repository
2. Open project folder
3. Install dependencies

npm install

4. Start development server

npm run dev

5. Open in browser

http://localhost:3000

---

## Folder Structure

app/ → Website pages and layout
components/ → Reusable UI components
public/ → Images and static assets
styles/ → Global styling
